Project set-up : 1. Create a network/hotspot using router or phone (other than server phone) 2. Connect the server phone to the hotspot 3. Connect the server laptop to the hotspot 4. Find the IP of the server phone & Make the changes in the serverToAndroidCode.php file. Note : Port number is default 7881, if changed,then the changes has to be made on both the server devices ie. server laptop & server phone(Android code) 5. Find IP of the server laptop by using cmd ipconfig & then enter it on the server phone android app. 6. Two way communication is established.
